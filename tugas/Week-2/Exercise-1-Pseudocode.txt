2. Tahun Kabisat
//  Pseudocode
Store "a" with data 4
Store "b" with data 100
Store "c" with data 400
store "Year" without any value
IF "Year" modulo "a" equal 0 and "Year" modulo "b" not equal to 0 OR "year" modulo "a" equal 0 and "year" modulo "b" equal 0 and "year" modulo "c" equal 0
    DISPLAY "ini tahun kabisat"
Else
    DISPLAY "ini bukan tahun kabisat"

//  Nature Language
Simpan "a" dengan data 4
Simpan "b" dengan data 100
Simpan "c" dengan data 400
Simpan "Year" tanpa data
Jika "Year" habis dibagi 4 dan tidak habis dibagi 100 ATAU "year" habis dibagi 4 habis dibagi 100 habis dibagi 400,
maka lakukan step 2a. Lakukan "ini tahun kabisat". Selain itu lakukan step 2b. Lakukan "ini bukan tahun kabisat"

3. Laundy Day
//  Pseudocode
Store "start machine" with 0
WHILE "start machine" lesser than 20
    DISPLAY "kelas belum penuh" 
    ADD "start machine" with by 1
END OF WHILE
DISPLAY "Mulai mencuci"

//  Nature Language
Simpan "start machine" dengan nilai 0
Ketika "start machine" masih dibawah 20 Tampilkan "kelas belum penuh"
Ulangi step 4 ketika "start machine" masih dibawah 20 dan tambahkan 1 pada "start machine"
tampilkan "mulai mencuci" jika "start machine" sudah 20

4. Periksa Kuku
//  Pseudocode
Store "a" with 0
WHILE "a" lesser than 40
    IF "a" equal "kuku panjang"
        DISPLAY "mendapat pujian"
    ELSE
        DISPLAY "mendapat hukuman"
    ADD "a" by 1
END OF WHILE

//  Nature Language
